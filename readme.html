Quick Start

Prerequisites

  • x86_64-elf-gcc (Cross-compiler)
  • NASM (Netwide Assembler)
  • QEMU (For testing and emulation)
  • GRUB2 (For creating bootable ISO)

Building

# Clone the repository git clone https://github.com/sandiphembram2021/kronos-operating-system.git cd kronos-operating-system # Build the kernel make # Create bootable ISO make iso # Run in QEMU make run

Installation on Real Hardware

  1. Build the ISO file using the commands above
  2. Flash the ISO to a USB drive using Rufus or Balena Etcher
  3. Boot your computer from the USB drive
  4. Type gui to launch the desktop environment

Shell Commands

Once booted, you'll see the Kronos shell prompt. Available commands:

System Commands

  • help - Show available commands
  • clear - Clear the screen
  • meminfo - Display memory usage statistics
  • uptime - Show system uptime
  • echo <text> - Echo text to console
  • reboot - Restart the system
  • halt - Halt the system

GUI Commands

  • gui - Start the graphical user interface
  • desktop - Launch desktop environment
  • demo - Show interactive GUI demonstration

System Requirements

  • CPU: 64-bit x86_64 processor
  • RAM: 512MB minimum (1GB recommended)
  • Storage: 1GB free space for installation
  • BIOS: Legacy BIOS support required
  • Boot: USB bootable device

Development

Kronos OS is built with modern development practices:

  • Language: C and x86_64 Assembly
  • Architecture: Monolithic kernel design
  • Build System: GNU Make with custom scripts
  • Testing: QEMU-based virtual testing
  • Debugging: GDB integration for kernel debugging

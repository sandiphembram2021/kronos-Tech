Install Kronos OS on Your PC

1

Download & Build Kronos OS

Download the source code, install build tools (GCC, NASM, QEMU), then run make && make iso to create your bootable ISO file.

2

Create Bootable USB Drive

Use Balena Etcher, Rufus, or dd command to flash the generated kronos.iso to a USB drive (minimum 1GB).

3

Boot from USB on Your PC

Restart your computer, enter BIOS/UEFI settings (F2/F12/Del), set USB as first boot device, and boot from the USB drive.

4

Launch Desktop Environment

Kronos OS will boot to command line. Type gui to launch the full desktop environment with applications, file manager, and terminal.

How to Contribute

Getting Started with Contributions

Contributing to Kronos OS is a great way to learn about operating system development while helping improve the project. Here's how you can get involved:

1

Fork the Repository

Create your own copy of the Kronos OS repository on GitHub

2

Set Up Development

Install build tools and set up your development environment

3

Find an Issue

Look for "good first issue" labels or create your own improvements

4

Make Changes

Implement your feature or fix, following coding standards

5

Test Thoroughly

Test your changes in QEMU and document any new features

6

Submit Pull Request

Create a pull request with a clear description of your changes

Installing Kronos OS on Real Hardware

This guide will walk you through the process of installing and running Kronos OS on actual computer hardware. Please read all warnings and requirements before proceeding.

Important Warning: Installing an operating system can potentially damage your computer or cause data loss. Always backup your important data and proceed at your own risk. This is experimental software.

Prerequisites

Hardware Requirements

  • CPU: 64-bit x86_64 processor (Intel/AMD)
  • RAM: Minimum 512MB, recommended 1GB or more
  • Storage: USB drive (minimum 1GB) for bootable media
  • BIOS: Legacy BIOS support (UEFI not yet supported)
  • Graphics: VGA-compatible display adapter
  • Input: PS/2 or USB keyboard and mouse

Software Requirements

  • USB flashing tool (Rufus, Balena Etcher, or dd command)
  • Built Kronos OS ISO file
  • Access to computer BIOS/UEFI settings

Step-by-Step Installation

1 Build Kronos OS

First, build the Kronos OS ISO file from source code:

git clone https://github.com/sandiphembram2021/kronos-operating-system.git cd kronos-operating-system make && make iso

This creates kronos.iso in your build directory.

2 Prepare USB Drive

Create a bootable USB drive using your preferred tool:

Using Rufus (Windows):

  1. Download Rufus from rufus.ie
  2. Select your USB drive
  3. Choose the kronos.iso file
  4. Click "START"

Using dd (Linux/macOS):

sudo dd if=kronos.iso of=/dev/sdX bs=4M status=progress # Replace /dev/sdX with your USB device

3 Configure BIOS

Boot your computer and enter BIOS setup:

  • Press F2, F12, Del, or Esc during startup
  • Navigate to Boot settings
  • Set USB as first boot device
  • Disable Secure Boot if present
  • Enable Legacy BIOS mode
  • Save and exit

4 Boot Kronos OS

Insert the USB drive and restart your computer:

  • Computer should boot from USB
  • GRUB bootloader will appear
  • Select "Kronos OS" from menu
  • Wait for kernel to load
  • You'll see the Kronos shell prompt

5 Launch GUI

Once booted to the shell, start the desktop:

kronos> gui

This will launch the full desktop environment with:

  • Desktop with wallpaper
  • Taskbar with start menu
  • Window manager
  • Built-in applications

6 Explore System

Try the available applications and features:

  • Terminal emulator
  • File manager
  • Text editor
  • Calculator
  • Games
  • System settings

Known Issues

  • UEFI Systems: May not boot without Legacy BIOS mode
  • Secure Boot: Must be disabled for installation
  • WiFi: No wireless network support yet
  • Audio: Sound drivers not implemented
  • USB 3.0: Limited USB support
  • Modern Graphics: No GPU acceleration

Troubleshooting

Boot Issues

System won't boot from USB:
  • Verify USB drive is properly flashed
  • Check BIOS boot order settings
  • Try different USB ports
  • Disable Secure Boot
  • Enable Legacy BIOS mode

Kernel Panic

System crashes during boot:
  • Hardware may not be compatible
  • Try different computer
  • Check memory requirements
  • Report issue on GitHub

Display Problems

No display or garbled output:
  • Ensure VGA-compatible graphics
  • Try different monitor/cable
  • Check display resolution settings
  • Some modern graphics may not work

Safety Considerations

Important Safety Notes:
  • Kronos OS is experimental software
  • Always backup important data before testing
  • Use on test machines when possible
  • Monitor system temperature during use
  • Have recovery media available

Recommended Test Systems

Ideal Hardware for Testing

  • Older Desktop PCs: 2010-2015 era systems work best
  • Virtual Machines: VirtualBox or VMware for safe testing
  • Dedicated Test Machine: Computer you can afford to experiment with
  • Standard Components: Avoid exotic or very new hardware

What to Avoid

  • Primary work computers
  • Systems with important data
  • UEFI-only modern laptops
  • Systems with custom firmware

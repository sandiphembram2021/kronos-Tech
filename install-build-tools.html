Ubuntu/Debian Setup

1. Update Package Manager

First, update your package lists to ensure you get the latest versions:

sudo apt update
sudo apt upgrade -y

2. Install Basic Build Tools

Install essential development tools and libraries:

sudo apt install -y build-essential git curl wget

3. Install NASM (Assembler)

NASM is required to assemble the assembly language parts of Kronos OS:

sudo apt install -y nasm

4. Install QEMU (Emulator)

QEMU is used for testing Kronos OS in a virtual environment:

sudo apt install -y qemu-system-x86

5. Install Cross-Compiler (GCC)

Install the cross-compiler for x86_64 target:

# Option 1: Use system GCC (simpler)
sudo apt install -y gcc

# Option 2: Install cross-compiler (recommended for OS development)
sudo apt install -y gcc-multilib
Note: For serious OS development, consider building a proper cross-compiler. See the advanced section below.

6. Install GRUB Tools

GRUB tools are needed to create bootable ISO files:

sudo apt install -y grub-pc-bin grub-common xorriso
Verification: Test your installation by running: 
nasm -v && gcc --version && qemu-system-x86_64 --version

Windows Setup

1. Install MSYS2

MSYS2 provides a Unix-like environment on Windows:

  1. Download MSYS2 from https://www.msys2.org/
  2. Run the installer and follow the setup wizard
  3. Open MSYS2 terminal and update the package database:
pacman -Syu

2. Install Build Tools in MSYS2

Install the required development tools:

pacman -S mingw-w64-x86_64-gcc
pacman -S mingw-w64-x86_64-nasm
pacman -S make
pacman -S git

3. Install QEMU

Download and install QEMU for Windows:

  1. Download QEMU from https://www.qemu.org/download/#windows
  2. Run the installer
  3. Add QEMU to your system PATH

4. Alternative: WSL2 (Recommended)

For a better Linux-like experience, use Windows Subsystem for Linux:

# In PowerShell (as Administrator)
wsl --install -d Ubuntu

# Then follow Ubuntu instructions above
Note: WSL2 with Ubuntu is often easier than native Windows tools for OS development.

macOS Setup

1. Install Homebrew

Homebrew is the package manager for macOS:

/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"

2. Install Xcode Command Line Tools

Install Apple's development tools:

xcode-select --install

3. Install NASM

brew install nasm

4. Install QEMU

brew install qemu

5. Install Cross-Compiler Tools

brew install x86_64-elf-gcc
brew install x86_64-elf-binutils

Arch Linux Setup

1. Update System

sudo pacman -Syu

2. Install Build Tools

sudo pacman -S base-devel git

3. Install NASM

sudo pacman -S nasm

4. Install QEMU

sudo pacman -S qemu-system-x86

5. Install GRUB Tools

sudo pacman -S grub xorriso

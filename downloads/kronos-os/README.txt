Kronos OS Downloads Directory
=============================

This directory contains all downloadable files for Kronos OS.

Available Files:
================

Main Release:
- kronos-v1.0.iso (Main bootable ISO file)
- kronos-v1.0.sha256 (Checksum for verification)

Virtual Machine:
- kronos-v1.0.ova (VirtualBox/VMware image)

Documentation:
- kronos-docs-v1.0.pdf (User manual and documentation)

Development:
- kronos-sdk-v1.0.tar.gz (Software Development Kit)

Previous Versions:
- kronos-v0.9-beta.iso (Beta release)
- kronos-v0.8-alpha.iso (Alpha release)
- kronos-v0.9-changelog.txt (Beta changelog)
- kronos-v0.8-changelog.txt (Alpha changelog)

File Verification:
==================
Always verify downloaded files using the provided SHA256 checksums:

Windows: certutil -hashfile filename.iso SHA256
Linux/Mac: sha256sum filename.iso
